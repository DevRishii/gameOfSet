This is the Game of Set

To start playing the game, download the file and open the file path in terminal.
Run "ruby main.rb" in the file path

The game allows you to have an infinite number of players along with an infinite number of bots.
The game will end once there are no more cards left in the deck or if the total number of points has reached 27.

You receive points by creating a set of 3 cards.

A set is defined as follows:
    They all have the same number or have three different numbers.
    They all have the same shape or have three different shapes.
    They all have the same shading or have three different shadings.
    They all have the same color or have three different colors.

Once the game ends you will have an option to either start a new game or stop playing.
Enjoy!
